# CRM Agent Workspace

This workspace contains the durable instructions and helper scripts used by the CRM scheduled agents.

It is not a source of runtime truth.

Runtime state lives in:

- the CRM database,
- CRM process tasks,
- CRM emails,
- CRM agent messages,
- CRM process definitions and draft versions.

Use this workspace for:

- system prompts,
- API documentation,
- helper scripts,
- safety rules,
- shared conventions.
